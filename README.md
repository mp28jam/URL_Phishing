# URL Phishing Detector

A machine learning application that detects phishing URLs using logistic regression and supervised learning classification. 

## Required installations
Ensure you have python installed on your machine by downloading:
```bash
python.org
```
Install the packages with:
```bash
pip install -r requirements.txt
```

- Python 3.x
- Flask
- pandas
- scikit-learn
- NumPy
- SciPy
- joblib


## Usage

1. Run the application:
```bash
   python app.py
```

2. Open your browser and navigate to:
```
   http://localhost:5000
```

3. Enter a URL to check if it's malicious or safe.
